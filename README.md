# 💧 Water Quality Analysis — Potability Prediction

## KIIT Mini Project | School of Computer Engineering

### About
Predicts whether water is safe to drink based on 9 physicochemical 
parameters using Machine Learning.

### Dataset
Kaggle Water Potability Dataset — 3,276 samples, 9 features

### Algorithms Used
- Logistic Regression — 59.55%
- K-Nearest Neighbours — 62.53%
- Random Forest — 69.73% ✅ Best

### Tech Stack
Python · Flask · scikit-learn · pandas · HTML/CSS/JS

### How to Run
pip install flask scikit-learn pandas numpy
cd webapp
py app.py
Open http://localhost:5000
